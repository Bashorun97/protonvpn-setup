<ul>
<h2>How To </h2>
<li>
Clone this repository
</li>
<li>
Make setup.sh file executable by running <code>chmod 755 setup.sh"</code>
</li>
<li>
Run setup.sh in root mode <code>sudo ./setup.sh</code>
